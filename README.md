# Платформа психологической поддержки жертв кибер-буллинга

Учебный мини-проект: веб-приложение с чат-ботом-консультантом, который поддерживает пользователей, столкнувшихся с кибер-буллингом. Бот работает на локальной модели Ollama `qwen2.5:7b` через HTTP API и отвечает в нейтральном, поддерживающем тоне.

## Архитектура

- **Frontend**: одна HTML-страница с чатом, отправляет историю диалога на backend.
- **Backend**: Flask-приложение (`app.py`) принимает сообщения, формирует системный prompt, отправляет запрос в локальную Ollama и возвращает ответ модели.
- **LLM**: локальная Ollama (`qwen2.5:7b`) через `http://localhost:11434/api/generate`.

## Требования

- Python 3.10+
- Установленная и запущенная Ollama
- Загруженная модель `qwen2.5:7b`

## Запуск

1. Убедитесь, что Ollama запущена и модель загружена:
   ```bash
   ollama run qwen2.5:7b
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   python app.py
   ```
4. Откройте в браузере: `http://localhost:5000`
